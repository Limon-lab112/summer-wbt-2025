  • School Management System

    Designed and developed a C#-based School Management System with three user roles: Admin, Teacher, and Student. Teachers can assign courses, submit results, share homework, notes, and send notifications. Students can view academic results, access notes, ask for counseling, check routines, and find classroom locations. Admins have full control to update course availability, modify course fees, manage academic calendars, and broadcast notifications to both teachers and students. The system follows modular and user-centric design to ensure efficiency in academic operations.

  • Smart HealthCare System with AI-based Symptom Checker

    Designed and developed a desktop application for smart healthcare management, featuring role-based access for Patients, Doctors, and Admins. The system includes an AI-powered symptom checker that uses a machine learning model to provide preliminary diagnoses based on user input. Patients can book appointments, receive prescriptions, view medical history, and get automated health reminders. Doctors can manage appointments, update treatment plans, and track patient records. The application simulates real-time health data and provides printable medical reports. Emphasizes OOP, modular architecture, and efficient healthcare workflow automation.
